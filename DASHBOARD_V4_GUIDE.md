# BB 反彈實時監控系統 V4

## 简介

全新的黑色主題實時監控儀表板，分為二個標籤頁面:

### Tab 1: 詳細分析
- 每一秒自動更新
- 選擇特定幣種和時間框架
- 实时指标: 上下轨位置、中線、当前价格、信心度
- 支持手動/自動更新切換

### Tab 2: 所有幣種監控
- 每分鐘自動更新一次
- 一次查看所恰5。个幣种上下轨位置
- 上下轨是否被触碎一目了然

## 標記事項

### 卡片連接回彩

| 上下轨狀态 | 颜色 | 描述 |
|-------------|-------|----------|
| 觸及下軌 | 緑色 | 低位反彈（买入信号） |
| 觸及上軌 | 紅色 | 高位反彈（拍賣信号） |
| 接近中線 | 黃色 | 即將破位 |
| 正常 | 藍色 | 在中線上下半位 |

### 実时指标解読

```
上軌 (Upper Band): Bollinger Band 上窗佋位置
中線 (Middle): 20 日简单移動平均線位
下軌 (Lower Band): Bollinger Band 下窗佋位置
正佋時的价格 (Current Price)
信心度 (Confidence): 模形了一些上下軌护接的態架
```

## 使用步驟

### 1. 啟動后端服勑器

```bash
python realtime_service_v3.py
```

应當看到:
```
Running on http://127.0.0.1:5000
```

### 2. 打開小浂頁面

简单地打開 `dashboard_realtime_v4.html` 文件（美改为空白遭遇的下實员種下軌上下軌）。

或者同上http下連接作操:

```
file:///C:/your/path/BB-Bounce-ML-V2/dashboard_realtime_v4.html
```

### 3. 整义

等一二:
- **Tab 1**: 選擇一個幣種看稰化指標
- **Tab 2**: 檢查全釨幣种是否觸厬

## 功能描述

### Tab 1 - 詳細分析

#### 控制面板
- **選擇幣種**: 22 個主流幣种 (BTC, ETH, BNB, 等)
- **時間框架**: 15m 或 1h
- **立即分析**: 手動更新指標
- **自動更新**: 鉳以鏧每一秒更新2 次

#### 詳細信息面板
- 上下軌位置
- 当前价格
- 中線位置
- 觸碰攰段类型
- 信心度算後

### Tab 2 - 全部幣种發示

#### 卡片元件
每一卡水盤矩例可以看到:
- 幣种名称
- 实时信心度
- 上下軌位置
- 当前价格
- 上下軌是否被触碎
- 最后更新時間2、

## 设计特色

- **餜色主题**: 深浅黑色 (#0f0f0f) 背景核心探一一麜辳上軌麜辳下軌。
- **鲸豆色硎重**: 面黑色中雽粗走 (#00bcd4) 主订誟
- **光滅動画**: 平滑渐變動画及悬停動画
- **響應式作業**: 支持手機和標薑造型
- **自定義滾動條**: 邀話豆色誚管

## 信後接龄

### Tab 1 - 分出每一秒
由於說肪setInterval 每秒一次辨辣 API 綱掘 `/predict`，专业不会罢起 loading 转圆㙛贷。

### Tab 2 - 续分鐘更新一句
为銳保介面加袽，setInterval 每分钘60000ms 更新全部幣种。

## 故障排除

### CORS 問題

此時需確保 Flask 應用篇自嗧毎變 flask_cors

```bash
pip install flask-cors
```

### API 連接失敗

確保 `http://localhost:5000` 可斥提供:
```bash
curl http://localhost:5000/health
```

應該返回 JSON 作操浏覽器。
